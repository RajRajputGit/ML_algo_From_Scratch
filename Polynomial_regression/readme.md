# Polynomial Regression

A simple implementation of Polynomial Regression to model relationships between variables when the pattern is non-linear.

Instead of fitting a straight line (like Linear Regression), polynomial regression fits a curved line that better represents complex patterns in the data.

What is Polynomial Regression?

Polynomial Regression is a machine learning technique used when the relationship between the input (X) and output (y) is curved instead of linear.

For example:

Linear regression: straight line

Polynomial regression: curve

General formula:
y = b0 + b1x + b2x² + b3x³ + ... + bnxⁿ

Where:

y → predicted value

x → input feature

b0, b1, b2... → coefficients

n → degree of the polynomial

# Why Use Polynomial Regression?

Sometimes data does not follow a straight line.

Polynomial regression helps when:

Data shows curved patterns

Linear regression performs poorly

You want to model non-linear relationships

Example use cases:

Growth trends

Temperature changes over time

Economic data

Population prediction
