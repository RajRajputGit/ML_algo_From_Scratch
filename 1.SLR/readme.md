## 🧠 What is Simple Linear Regression?

Simple Linear Regression models the relationship between:

* **One independent variable (X)**
* **One dependent variable (Y)**

It assumes a linear relationship of the form:

[
y = \beta_0 + \beta_1 x
]

Where:

* **β₀** → Intercept
* **β₁** → Slope (Coefficient)
* **x** → Independent variable
* **y** → Predicted output

## 📊 Model Interpretation

* **Coefficient (Slope):** Indicates how much the dependent variable changes for one unit change in the independent variable.
* **Intercept:** The predicted value of the dependent variable when the independent variable is 0.
